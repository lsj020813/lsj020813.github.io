# LEESANGJUN Blog

GitHub Pages blog powered by [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy).

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

The local preview runs at <http://127.0.0.1:4000>.

## Deployment

Push to `master`. The GitHub Actions workflow in `.github/workflows/pages-deploy.yml` builds and deploys the site to GitHub Pages.
